Instalacao em minutos

Comece a usar no Chrome ou Edge

  1. Baixe ou clone o projeto da extensao e execute npm install.
  2. Rode npm run build para gerar os arquivos em dist.
  3. Abra chrome://extensions ou edge://extensions.
  4. Ative modo desenvolvedor e carregue a pasta dist.
